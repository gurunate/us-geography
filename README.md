# US Geography

## Getting Started

### Install

```
$ yarn
```

### Run Quiz App

```
$ cd quiz-app
$ yarn dev
```

http://localhost:3123

### Run Data Graph

```
$ cd data-graph
$ yarn dev
```

http://localhost:3123/graphql
